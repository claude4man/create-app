# Changelog

Все заметные изменения в этом проекте документируются в этом файле.

## [1.0.0] - 2026-03-28

### ✨ Добавлено
- Первоначальный релиз create-app CLI
- Поддержка выбора компонентов (Backend, Frontend, Admin, Landing)
- Динамическая генерация docker-compose.yml
- Автоматическая загрузка последних версий npm пакетов
- GitHub Actions workflows для автоматического развертывания
- Поддержка опциональной PostgreSQL базы данных
- Индивидуальная настройка портов для каждого сервиса
- TypeScript/Prisma для NestJS backend
- Next.js frontend и admin панель с MUI
- Автоматическое создание .env файлов

### 🔧 Изменено
- Улучшена загрузка версий пакетов (теперь параллельная)
- Оптимизированы docker-compose конфигурации

### 🐛 Исправлено
- PostgreSQL больше не включается если не выбрана
- Правильное отображение портов в документации
- Исправлена обработка компонентов в зависимостях

### 📚 Документация
- Добавлен подробный README
- Создан CONTRIBUTING.md для разработчиков
- Добавлена лицензия MIT
